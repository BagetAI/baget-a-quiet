# A Quiet: High-Fidelity Listening Bar & Record Shop

A sanctuary for the active listener in the heart of Tel Aviv (Lev Hair / Montefiore).

## Features
- **Editorial Experience**: A high-end, asymmetric layout designed for design-conscious audiophiles.
- **Live Inventory**: Dynamic record gallery pulling from the "A Quiet Inventory" database.
- **Genre Filtering**: Browse curated Jazz and Soul selections.
- **The Quiet Circle**: Integrated membership waitlist and notification system.
- **Audiophile Rig**: Technical specs featuring Technics, Leben, and Klipsch hardware.

## Tech Stack
- Vanilla JavaScript / HTML5 / CSS3
- Agent Database API for dynamic inventory and lead capture
- Brand Palette: Aged Parchment (#F5F2E9), Deep Walnut (#3C2F2F), Tube Glow Orange (#FF8C00).
